# 电子族谱系统 - 本地版

纯前端实现的电子族谱管理系统，无需安装任何依赖，直接用浏览器打开即可使用。

## 🚀 快速开始

1. **双击打开 `index.html`** 文件
2. 浏览器会自动加载应用
3. 开始添加家族成员和关系

## ✨ 功能特性

### 📊 关系图谱
- D3.js 力导向图可视化
- 支持缩放、拖拽节点调整布局
- 点击节点查看成员详情
- 不同性别用不同颜色区分（蓝色=男性，粉色=女性，灰色=已故）

### 👨‍👩‍👧‍👦 成员管理
- 添加/编辑/删除家族成员
- 成员属性：姓名、性别、出生日期、辈分、分支、状态
- 成员卡片展示，网格布局

### 🔗 关系管理
- 支持多种关系类型（父子、母子、夫妻、兄弟姐妹等）
- 自动创建反向关系
- 关系连线带标签显示

### 💾 数据存储
- 使用浏览器 localStorage 持久化
- 支持导出 JSON 备份
- 支持导入 JSON 恢复
- 支持清空数据

## ⚠️ 注意事项

- **数据存储在浏览器本地**，清除浏览器数据或更换浏览器会导致数据丢失
- **重要：请定期使用「数据管理」功能导出 JSON 备份**
- 建议使用 Chrome、Firefox、Edge 等现代浏览器

## 📱 界面说明

### 顶部导航
- **关系图**：显示家族关系网络图
- **成员管理**：以卡片列表形式管理所有成员
- **数据管理**：导入/导出/清空数据

### 操作按钮
- **+ 添加成员**：新增家族成员
- **添加关系**：建立两个成员间的关系

### 侧边面板
- 点击成员节点或卡片可打开详情面板
- 显示成员基本信息
- 显示该成员的所有关系列表

## 🎨 颜色说明

| 颜色 | 含义 |
|------|------|
| 🔵 蓝色 | 男性成员 |
| 🩷 粉色 | 女性成员 |
| ⚫ 灰色 | 已故成员 |

## 📄 文件结构

```
local/
└── index.html    # 完整的单文件应用（约43KB）
```

## 🔧 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **D3.js v7** - 数据可视化库
- **Element Plus** - UI 组件库
- **localStorage** - 本地数据持久化

所有依赖均通过 CDN 加载，无需安装任何软件。

---

*本版本为纯前端实现，数据仅存储在浏览器本地。如需多设备同步或在线协作，请使用完整版（需要后端服务器）。*
